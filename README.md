# Bus Search App 簡易公車查詢系統

## 介紹
介接 [TDX 運輸資料流通服務 API](https://tdx.transportdata.tw/) 資料，實作呈現公車資訊。

![](https://imgur.com/AmdebcT)

[Live Demo](https://a3216lucy.github.io/bus_search_app/)

## 功能
- 可進行全臺縣市，進行公車資訊查詢
- 即時顯示公車到站時間、站牌名稱、車牌

## 技術學習
- 使用 TailWindCSS + JavaScript 來撰寫
- 透過 DOM 控制網頁內容
- 使用 axios 串接外部 API
